Installation
============

1. Navigate to **Apps**.
2. Search with keyword *to_hr_payroll_meal*.
3. Press **Install**.

Instruction
===========

**Instruction video:** `HR Meal Order & Payroll Integration <https://youtu.be/AG-kmRg2BMg>`_

**1. Set the price to be paid to employees**

Navigate to **Payroll ‣ Configuration ‣ Settings**, then navigate to the *Employee Meal Price* field to add the default price:

.. image:: 01-thiet-lap-don-gia-suat-an-cua-nhan-vien.en.jpg
   :align: center
   :height: 550
   :width: 1100
   :alt: Set up the Employee Meal Price

In addition, you need to set up a meal price by navigating to **HR Meal ‣ Configuration ‣ Meal Type**, select a meal type and enter the price for them:

.. image:: 02-thiet-lap-don-gia-suat-an.en.jpg
   :align: center
   :height: 280
   :width: 1100
   :alt: Set up the price for the meal order

**2. Create a Meal Order**

Navigate to **HR Meal ‣ Meal Orders ‣ Draft Orders** to create a new meal order, confirm and approve the order. Here, you will see information about the meal value that employees and the company have to pay:

.. image:: 03-don-gia-suat-an-tren-phieu-dat-suat-an.en.jpg
   :align: center
   :height: 550
   :width: 1100
   :alt: Meal Order

**3. The Salary Computation**

Navigate to **Payroll ‣ Payslips ‣ Payslips** to create payslips for employees (or **Payroll ‣ Payslips ‣ Payslips Batches** to batch generate payslips for employees). When press *Compute Sheet*, on payslips, you will see the information:
