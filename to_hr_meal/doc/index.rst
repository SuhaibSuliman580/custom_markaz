Installation
============

* Navigate to **Apps**.
* Search with keyword *to_hr_meal*.
* Press **Install**.

Instruction
===========

**Instruction video:** `Meal Orders <https://youtu.be/a9VQ9qPMAvE>`_

**1. Initial configuration**

To start using the Meal management software, you need to do some initial configuration by going to the **HR Meal > Configuration**. The configuration includes:

* Create HR Meal Tyles: create meal styles to categorize meals by criteria such as time or price. For example lunch, dinner, normal meal, deluxe meal,...

.. image:: 01-tao-kieu-suat-an.en.jpg
   :alt: Create HR Meal Tyles
   :width: 1100
   :height: 300
   
* Create HR Kitchen: create the company's internal kitchens (if necessary) to manage the company's kitchens.

.. image:: 02-tao-bep-an.en.jpg
   :alt: Create HR Kitchen
   :width: 1100
   :height: 250

* Create HR Meal Type Alerts: create notifications for meal types, to alert bookers about the time to order meals. For example, lunch must be ordered by 10 a.m.

.. image:: 03-tao-canh-bao-kieu-suat-an.en.jpg
   :alt: Create HR Meal Type Alerts
   :width: 1100
   :height: 300
   
**2. Create HR Meal orders**

* The responsible person will go to **HR Meal > HR Meal Orders > Create** to create meal orders. Here, you need to fill in the following information:

  * Meal type of the meal order;
  * Meal order object: you can create meal orders for all employees, specific departments or employees;
  * Meal order source: from the company's internal kitchen or an external supplier.

In addition, you can also order meals for guests corresponding to the staff in charge when needed.

.. image:: 04-tao-don-dat-suat-an.en.jpg
   :alt: Create HR Meal orders
   :width: 1100
   :height: 500
   
Click **Confirm** after completing the above information.

* The meal manager checks the information and processes **Approve** or **Refuse** the meal order to finalize the number of meals to notify the kitchen or supplier.

.. image:: 05-phe-duyet-don-dat-suat-an.en.jpg
   :alt: Approve the meal order
   :width: 1100
   :height: 550
   
**3. Meal orders analysis**

Go to **HR meal > Reports > Meal orders analysis** to view meal reports for your management purposes. For example: When is the time to order meals; How many meals did each kitchen serve to plan personnel and materials; Why is the trend of increasing/decreasing the cost of meals,...

.. image:: 06-bao-cao-suat-an.en.jpg
   :alt: View Meal orders analysis
   :width: 1100
   :height: 350
