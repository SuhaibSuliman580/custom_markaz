
Installation
============

1. Navigate to **Apps**;
2. Search with keyword *to_hr_payroll*;
3. Press **Install**.

Instruction
===========

**Instruction video:** `Payroll <https://youtu.be/VQNphmz9PT0>`_

**1. Initial configurations for payroll**

* `Salary cycle <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/salary-cycle.html>`_: The salary cycle of the business. The default will be from the beginning to the end of the current month. You can create a new one in case your company has a different Salary cycle. Example: Salary cycle from the 25th of this month to the next 25th.

* `Salary Structure <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/salary-structures.html#salary-structures>`_: A salary structure is a set of rules used to determine how much employees will be paid. The system allows the creation of many salary structures such as salary structure for the office department, salary structure for the production department, etc.

* `Salary rule <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/salary-rules-and-salary-rule-categories.html#salary-rule-configuration>`_: Salary rules are also understood as calculation formulas for items on payslips. The system has automatically added a set of basic salary rules.

.. image:: 01-quy-tac-luong.en.jpg
   :align: center
   :height: 500
   :width: 1100
   :alt: Salary rules
   
* `Personal tax rule <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/personal-tax-rules.html#modify-the-personal-tax-rules>`_: Rules for calculating Personal Income Tax.

.. image:: 02-quy-tac-thue-tncn.en.jpg
   :align: center
   :height: 200
   :width: 1100
   :alt: Personal tax rule
   
* `HR Advantage Templates <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/hr-advantage-templates.html#hr-advantage-templates>`_: Allowances for employees according to company policy. For example Harmful Subsidies, Hard Working Award, Performance Award, etc.

.. image:: 03-phu-cap-che-do-dai-ngo.en.jpg
   :align: center
   :height: 320
   :width: 1100
   :alt: HR Advantage Templates
   
* `Payroll Contribution Register <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/payroll-contribution/payroll-contribution-register-insurances.html#payroll-contribution-register-insurances>`_: Deductions from salaries of Employees and Enterprises.

.. image:: 04-dang-ky-dong-gop-tu-luong.en.jpg
   :align: center
   :height: 400
   :width: 1100
   :alt: Payroll Contribution Register
  
* `Contribution Registers/Contribution Register Category <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/configuration/contribution-registers-and-contribution-registers-categories.html#create-a-new-contribution-register>`_: Helps to classify contributions by entities involved in salary formation such as Employees, insurance agencies, unions, etc.

**2. Set up information on employee contract**

After completing the configuration in step 1, you need to set this information on the employee contract:

Go to **Payroll ‣ Employees ‣ Contracts**, create a new `employee contract <https://viindoo.com/documentation/16.0/applications/human-resources/employee/operations/create-labour-contract.html#how-to-create-a-contract>`_:

.. image:: 05-hop-dong-lao-dong.en.jpg
   :align: center
   :height: 550
   :width: 1100
   :alt: Labor contract
   
**3. Create Payslips Batches & Payslips**

Go to **Payroll ‣ Payslips ‣ Payslips** to create `payslips <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/payslips/create-payslips.html#create-payslips>`_ for each employee:

.. image:: 06-tao-phieu-luong.en.jpg
   :align: center
   :height: 470
   :width: 1100
   :alt: Create Payslips
   
Or go to  **Payroll ‣ Payslips ‣ Payslips Batches** to create `payslips batches <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/payslips/create-payslips-batches.html#create-payslips-batches>`_ for departments or the company:

.. image:: 02-bao-cao-pivot-phan-tich-thue-thu-nhap-ca-nhan-viindoo.en.jpg
   :align: center
   :height: 540
   :width: 1100
   :alt: Create Payslips Batches
   
**4. View analysis report**

Go to **Payroll ‣ Reports ‣ Payroll Analysis** to view `payroll analysis report: <https://viindoo.com/documentation/16.0/applications/human-resources/payroll/reports/payroll-analysis.html#payroll-analysis>`_.

.. image:: 02-bao-cao-phan-tich-luong-dang-pivot-viindoo.en.jpg
   :align: center
   :height: 480

   :alt: Payroll analysis report
