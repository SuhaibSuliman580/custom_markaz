"For Easier Reporting and Grouping"

This module adds the concept of types for payslip batches in Payroll management.

Description

The objective of the module is to create types for payslip batches to be used in Payroll management. This module extends the functionality of the Payroll module and allows you to group or filter payslip batches by a specific type. This feature adds a field for selecting types on the payslip batches form, and it also adds a menu item on the configuration menu to define types for payslip batches.
